🕸️ Network Tools

🧬 Network Injection (Network_Injection.py)

This script updates a precomputed HDF5 similarity network when new sequence nodes have been added to the embedding set. Instead of recalculating alignments for all pairwise combinations (which scale quadratically as N^2), it identifies which sequence combinations are pre-existing and copies their alignment scores directly from the old network cache. It maps only the new sequence combinations to CPU processes for parallel alignment calculations and merges the results into a fully-populated, updated HDF5 network file.

⚙️ Parameters

Parameter Description
OLD_NETWORK The pre-existing HDF5 network containing calculated alignment scores for original sequence pairs.
NEW_EMBEDDINGS The newly updated HDF5 embedding database containing both original and newly injected sequence embeddings.
LOCAL_GAP_P The gap open/extend penalty applied during local Smith-Waterman alignment calculations.
GLOBAL_GAP_P The gap open/extend penalty applied during global Needleman-Wunsch alignment calculations.
WORKERS The number of CPU threads allocated for parallel alignment of new sequence pairs.
BATCH_SIZE The number of sequence pairs aligned in a single multiprocessing block before writing results to disk, minimizing VRAM and RAM footprint.

📤 Network Extraction (Network_Extraction.py)

This script extracts a subset network from a large master HDF5 network file based on a whitelist of target sequences. It avoids running any new pairwise sequence alignments. Instead, it reads the node names (headers) from the whitelist FASTA file, constructs an index mapping to map old node indices to new subset indices, filters out any edges where one or both participating sequences are missing from the whitelist, and saves the extracted sub-network (along with optional traceback alignment paths) into a smaller, standalone HDF5 network.

⚙️ Parameters

Parameter Description
INPUT_NET The path to the master HDF5 network file to extract connections from.
INPUT_FASTA The FASTA file defining the subset of sequence headers that should remain in the filtered network.
