📥 Embedding Tools

🧬 Embedding Injection (Embedding_Injection.py)

This script updates an existing HDF5 embedding database with new sequence additions from a FASTA file. It avoids the need to recalculate embeddings for the entire dataset by performing a set difference calculation to identify only the new sequence headers. It loads the corresponding protein language model (pLM) that was used to build the original database, generates embeddings only for the new sequences, and merges them with the pre-existing embedding vectors into a synchronized, combined HDF5 file matching the order of the new FASTA file.

⚙️ Parameters

Parameter Description
INPUT_EMBED The pre-existing HDF5 database containing original sequence embeddings.
INPUT_FASTA The new FASTA file containing all previous sequences plus the new sequences to be added.

📤 Embedding Extraction (Embedding_Extraction.py)

This script extracts a subset of sequence embeddings from a large master HDF5 embedding database based on a target list of sequence headers. Rather than executing expensive neural network inference to generate embeddings for a smaller subset of interest, this script extracts the existing vectors from the master database file and writes them, along with updated metadata, into a new, smaller HDF5 file.

⚙️ Parameters

Parameter Description
INPUT_EMBED The source HDF5 database containing master sequence embeddings to extract from.
INPUT_FASTA The target FASTA file (or text file) defining the subset of sequence headers to extract.
