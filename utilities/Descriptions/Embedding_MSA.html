🌳 Embedding MSA

🧬 Embedding Multiple Sequence Alignment (Embedding_MSA.py)

This script computes a Multiple Sequence Alignment (MSA) using protein language model embeddings. While traditional MSA tools (like MAFFT or MUSCLE) rely on amino acid substitution matrices, this tool aligns profiles using the average of high-dimensional residue embedding vectors. This captures structural similarity and allows accurate alignment of distantly related sequences with low sequence identity.

The progressive profile alignment is guided by an evolutionary consensus tree built from a pairwise sequence similarity network. The script automatically intersects sequences found across the input FASTA, embedding set, and network file to target only the common subset of sequences.

💡 Input Cases and Network Behaviors:
  • Dense Network Input: If the input network is fully populated (dense all-vs-all), the pairwise distance matrix is constructed directly from the stored alignment scores. Consensus neighbor-joining trees are generated using bootstrap replicates. This scenario provides the most accurate progressive tree topology.
  • Sparse Network Input: If the input network is sparse (only a subset of edges is computed), the script cannot construct a complete distance matrix directly. To solve this, it fits an Isotonic Regression model. It calculates the global cosine distances between sequence mean embeddings (dense vectors available for all sequences) and maps the known network alignment scores to these cosine distances. The fitted regression curve is then used to predict the missing alignment scores.
  • BLAST Network Input: If the network is generated from BLAST E-values (containing 'blast' in the filename), alignment scores are processed as -Log10(E-value). Normalization modes and score selections are disabled in the GUI as they are not applicable to statistical E-values.

⚙️ Parameters

Parameter Description
INPUT_FASTA The input sequence file (in FASTA format) containing the raw sequences to be aligned and written as the final MSA.
INPUT_EMBED The HDF5 embedding database containing the dense residue embeddings.
INPUT_NETWORK The HDF5 network file containing pairwise connectivity scores used to construct the guide tree.
SHOW_REGRESSION_PLOT Toggle to show the diagnostic Isotonic Regression plot when using a sparse network. This visualizes the fit between mean embedding cosine distances and actual alignment scores.
ALIGNMENT_SCORE Select whether to construct the guide tree using 'global' or 'local' connectivity scores from the network.
NORMALIZATION_MODE The normalization method applied to alignment scores (e.g., alignment_length, shorter_sequence, longer_sequence, average_sequence). Disabled for BLAST networks.
NUM_TREES The number of bootstrap replicate trees generated by adding random Gaussian noise to the distance matrix. Averaging these provides a robust consensus guide tree.
NOISE_SCALE The standard deviation of random noise applied to the distance matrix during bootstrap tree building. Helps resolve branching ambiguities.
GAP_OPEN The gap open penalty score. More negative values penalize opening new gaps more severely.
GAP_EXTEND The gap extend penalty score. More negative values penalize extending existing gaps.
WORKERS The number of CPU threads allocated for parallel bootstrap guide tree calculations.
SAFE_TEMP_DIR The temporary directory used to cache intermediate files and memory-mapped matrices.

📉 Sparse MSA Converter (Sparse_MSA_Converter.py)

This script converts standard aligned FASTA multiple sequence alignments (which are often very large and filled with redundant gap '-' characters) into space-saving SciPy Compressed Sparse Row (CSR) matrices. It records only the non-gap amino acid positions and maps characters to integer values, reducing the memory footprint by over 95 percent and enabling rapid load times inside the SSN viewer.

⚙️ Parameters

Parameter Description
CONVERT_ALL Toggle to convert all FASTA multiple alignments inside the input directory. If disabled, only the selected alignment file is converted.
INPUT_FASTA The standard multiple sequence alignment file (in FASTA format) to be compressed.
